Amazon Web Services (AWS)

Compute:

EC2 (Elastic Compute Cloud): Virtual servers in the cloud.

Lambda: Serverless computing to run code without provisioning servers.

Elastic Beanstalk: Easy-to-use service for deploying and scaling web applications and services.

Storage:

S3 (Simple Storage Service): Scalable object storage.

EBS (Elastic Block Store): Block storage for use with EC2.

Glacier: Low-cost cloud archive storage.
